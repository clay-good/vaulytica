## Deviations

- **No classifier sub-domain entry.** Routing to the filing playbooks is via `selectMatchCandidates`, which admits an extended playbook on its own `match_features` and is classifier-independent — so a 17th TF-IDF sub-domain was not added, avoiding recalibration of the 16-family classifier and re-baselining the v4 classification goldens. The "a brief classifies to the filing family, an NDA does not" requirement is met by the playbooks' filing-specific title/phrase features plus the `ADMIT_THRESHOLD` gate, verified live (brief → `appellate-brief`; NDA → unchanged) and pinned by the golden suites (all 1064 still pass).
- **Profiles live at `src/filing/profiles/*.json`**, not `profiles/courts/`, so they bundle for the browser and import for the CLI with no runtime fetch/fs (deterministic everywhere). Schema-validated at module load and by `court-profile.test.ts`.
- **`--court` dormancy is by conditional rule-set inclusion**, not `assertion_gate`: the FILE rules use the `applies_to_playbooks` gate (the framework requirement) and join the rule set only when a profile is selected. This keeps every non-filing hash byte-identical without changing the runner (proven by the isolation tests + all goldens unchanged).
- **Type-volume exclusion is conservative on flat documents.** Excludable-block word counts are deduped by section and capped per section, so a document without clear section structure subtracts nothing and falls back to the total count — the safe direction for a compliance check (flags for review rather than falsely clearing).
- **Tab picker deferred.** `--court` is fully wired through the CLI and the browser `PipelineOptions.filing`; a court-picker UI control in the tab (with re-run) is a follow-up. A matched brief already renders the filing scope-of-review statement in the tab noting a profile is required.
- **Court limit values need an attorney/citation verification pass** before relying on them for a real filing (they carry `retrieved_at` and are structured for the currency mechanism); the 9th Cir. and California numbers are best-known values pending confirmation.
