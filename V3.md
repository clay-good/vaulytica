# Vaulytica v3 — Short Index

v3 extends v2 with compliance-grade rule sets for BAA (HIPAA), DPA (GDPR/UK GDPR/US state),
NDA (DTSA/UTSA), MSA (deep), EU SCCs, UK IDTA, and adjacent documents. Same browser-only,
no-AI, no-telemetry posture as v2.
