Why Vaulytica?

A powerful, self-hosted Python CLI tool delivering enterprise-grade security, compliance, and IT automation for Google Workspace environments.

Comprehensive Scanning

13 specialized security scanners covering files, OAuth apps, users, groups, mobile devices, Chrome OS, Gmail, audit logs, calendars, Vault, shared drives, and more.

Advanced PII Detection

Industry-specific pattern detection with 20+ built-in detectors for SSNs, credit cards, bank accounts, passport numbers, and custom patterns tailored to your organization.

Automated Compliance

Generate GDPR, HIPAA, SOC2, PCI-DSS, FERPA, and FedRAMP compliance reports automatically. Executive-friendly HTML dashboards with charts and visualizations.

Real-Time Monitoring

Continuous health checks, Prometheus metrics, and automated alerts. Schedule scans and get notified instantly when security issues are detected.

User Lifecycle Management

Create, update, suspend, restore, and delete users with powerful bulk operations. Automated employee onboarding and offboarding workflows.

Automated Backups

Scheduled backups of users, groups, and organizational structure. Export data in multiple formats with comprehensive version control.

Workflow Automation

Automated workflows for PII detection alerts, external file sharing monitoring, employee offboarding, and custom security policies.

Multi-Domain Support

Manage multiple Google Workspace domains from a single installation. Perfect for organizations with multiple business units or subsidiaries.

Self-Hosted Security

Your data never leaves your infrastructure. Deploy on-premises or in your private cloud. Full control over sensitive security and compliance data.

How It Works

Get started in minutes with our simple setup process

1️

Clone & Install

Clone the repository and install dependencies with Poetry. Requires Python 3.9+ and takes less than 2 minutes.

2️

Configure Credentials

Set up a Google Cloud service account with domain-wide delegation. Add required OAuth scopes and configure your domain.

3️

Start Scanning

Run your first READ-ONLY security scan. All scanning commands are safe and never modify your Google Workspace data.
